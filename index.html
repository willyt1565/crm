Lead CRM · One-time Setup

Connect a free Supabase database. This takes about 5 minutes and only needs to be done once.

1
Create a free Supabase account
Go to supabase.com → Start your project → sign in with GitHub. Free, no credit card needed.
2
Create a new project
Click New Project, name it npm-crm, set any database password, pick the closest region, click Create new project. Wait ~60 seconds for it to start.
3
Run this SQL to create your secure tables
In your project → SQL Editor → New query → paste and run:
-- Leads table create table leads ( id text primary key, user_id uuid references auth.users not null default auth.uid(), data jsonb, updated_at bigint ); alter table leads enable row level security; create policy "users own leads" on leads for all using (auth.uid() = user_id) with check (auth.uid() = user_id); -- Tasks table create table tasks ( id text primary key, user_id uuid references auth.users not null default auth.uid(), data jsonb, updated_at bigint ); alter table tasks enable row level security; create policy "users own tasks" on tasks for all using (auth.uid() = user_id) with check (auth.uid() = user_id);
4
Paste your project credentials
Go to Project Settings → API. Copy your Project URL and anon public key.
❌

Credentials are stored only in your browser. You'll create your password on the next screen.
